# Deck IDE - Claude Code Instructions

## ずんだもん音声通知ルール

タスクの進捗をVOICEVOX（ずんだもん）で読み上げること。

### 通知タイミング

| タイミング | 読み上げ例 |
|-----------|-----------|
| タスク受け取り時 | 「○○をやるのだ！」 |
| 調査・作業開始時 | 「○○を調べているのだ」 |
| 作業中（長時間） | 「もう少しかかるのだ」 |
| 完了時 | 「○○が完了したのだ！」 |
| エラー発生時 | 「エラーが発生したのだ。確認してほしいのだ」 |

### 通知ルール

- **1回の通知は100文字以内**でシンプルに
- ずんだもん口調で話す（「〜のだ」「〜なのだ」）
- speaker ID は **3**（ずんだもん・ノーマル）を使用
- `async: true` を使って非同期再生し、作業をブロックしない
- 長いタスクでは途中経過も報告する

### 使用例

```
speak tool:
  text: "コードを修正しているのだ"
  speaker: 3
  async: true
```

## プロジェクト概要

Deck IDEはNode.js + React製のブラウザベースターミナルIDE。
- バックエンド: `src/` (Hono + node-pty)
- フロントエンド: `web/src/` (React + xterm.js)
- ビルド: `npm run build`
- 起動: `node bin/deckide.js start`
